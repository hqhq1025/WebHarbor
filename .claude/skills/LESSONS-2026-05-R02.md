## R02 9/10 站 bug 清单（待修）

### harvest.py
- 缺 `--refresh-meta` flag（旧 page 不会重抓 → R01/R02 patch 无可观测覆盖率）
- bot-wall titles 应集中常量 `KNOWN_BOT_WALL_TITLES`（AWS WAF / Datadome / PerimeterX / CF Turnstile / Akamai）
- bot_block 应自动 Exa fallback（现在只写 FALLBACK_NEEDED.md 让 agent 手补）
- title 漂移：JS 注入徽章覆盖 document.title（coursera `(4) New Messages!`）— 应 DOMContentLoaded 时抓 title
- 缺 global Akamai-aware fetcher：raw curl 全 403，Playwright 通，多工具应共享 browser session/cookies
- 缺 CF-Turnstile auto-recover（discogs 9/16 页 blocked）
- silent_redirect_then_block reason（drugs.com `/condition/`→302→Access Denied 案例）

### extract_facets.py
- **label vs value 倒置**（craigslist input value="141,142" 当 label，真 label 在 sibling）
- panel 子区不切（craigslist cars 一 group 79 options 合一坨）
- 把年份 "2010" 当 count（discogs decade option）
- SPA modal-filter 不会抠（eventbrite/compass/coursera）

### extract_image_urls.py
- 写 HTML-entity garbage URL `&amp;quot;`（discogs / coursera cookielaw banner）— 需 html.unescape srcset/style
- map tile 混 entity image（craigslist 案例）— 需 image_class 字段（tile/logo/upload/cdn/ad-pixel）
- 不区分广告 tracking pixel vs 真 image

### extract_sprites.py
- 漏 `<svg style=display:none><symbol id=...>` 内联雪碧图（coursera 案例）
- 漏外部 `<use xlink:href="/sprite.svg#id">` 引用（apple/compass/cambridge 案例）

### extract_icons.py
- 漏 inline SVG icon system（github 132 个 `octicon-*` / heroicons / lucide / bootstrap-icons / fa class / aria-label+svg）
- download_samples icon picker 碰 hot-link 整体 fail（fandom www 子域 403）

### extract_video_urls.py
- 仅扫 `<video>/<source>/iframe`，漏 Wistia / DASH .mpd / data-video-id / JS player 注入（espn / coursera / fandom signed URL）
- gvt1 signed URL 几小时失效（fandom）— 需 `expires_in_url` 标记

### extract_animations.py
- css-bg URL 仍允全 webp（待严化）

### extract_sitemap.py
- sitemapindex 3 层嵌套现仅展开 2 层（fandom 案例）— 需递归
- 202/404/406 仍计入 sitemaps_found（espn 202 + content-length=0 / github 406 需 Accept: application/xml）
- 全 403 时无 fallback（drugs Akamai）

### extract_code_blocks.py
- 大部分站是 script[src] 噪声（占 95%）；需 script vs 真 code 分类

### infer_cdn_pattern.py
- 不写 `_cdn_pattern.json` 文件（仅 stdout）— audit 取不到
- beacon 域噪声（bat.bing.com / google-analytics）应过滤
