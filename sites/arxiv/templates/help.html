Getting Started

Welcome to arXiv! Here are the basics to get you started:

  • Browse papers by subject from the homepage.
  • Search using the bar at the top of every page — search by title, author, abstract, or ID.
  • Cite papers using the BibTeX export on each paper's abstract page.
  • Subscribe to daily alerts for any subject category.
{% elif section == 'submit' %}

How to Submit an Article

Authors can submit papers to arXiv by creating an account, selecting a subject area, uploading a LaTeX source or PDF, and providing metadata (title, authors, abstract).

  1. Create an arXiv account (requires institutional email for first-time authors).
  2. Prepare your source files in LaTeX or PDF format.
  3. Upload and enter metadata on the submission form.
  4. Preview the compiled result.
  5. Submit for moderation — papers typically appear within one business day.
{% elif section == 'withdraw' %}

How to Withdraw an Article

If your submission has not yet been announced, you can withdraw it directly from your user page without leaving any public trace. Simply locate the pending submission in your queue and click the Delete button before the announcement deadline.

If the article has already been announced, you cannot delete it. Instead, submit a new version with a brief withdrawal note in the abstract field explaining the reason; the paper will remain in the archive but be clearly marked as withdrawn.

See also: submission guidelines.

{% elif section == 'non-english' %}

Non-English Submissions

arXiv accepts papers in any language, although English is strongly preferred so that the work reaches the widest possible audience. For non-English submissions, authors must provide a multi-language abstract: the abstract in the paper's primary language followed by an English translation.

The multiple abstracts should be separated by -- (two hyphens) on a line by itself. For example:

Résumé en français...

--

English abstract here...

The -- separator lets arXiv's metadata pipeline correctly index each language variant.

{% elif section == 'figures' %}

Figure Formats for Submissions

arXiv accepts the following figure formats in LaTeX submissions:

  • PDF — preferred for vector graphics (plots, diagrams).
  • PNG — preferred for photographs and raster images.
  • JPG / JPEG — acceptable for photographs; avoid for line art.
  • EPS — accepted but increasingly deprecated; converted to PDF.

SVG and GIF are not supported. TIFF is accepted only when converted to PNG during the build. Animated and interactive figures should be included as ancillary files.

Subscribe to Daily Listing Emails

arXiv offers free daily listing emails announcing new submissions in any subject category you choose.

  1. Create an arXiv account and log in.
  2. Navigate to Alerts from the user menu.
  3. Enter the category code (e.g. cs.AI, math.AT) and select daily as the frequency.
  4. Click Subscribe. You will start receiving a listing email each business day summarising new submissions.

You can manage or unsubscribe from alerts at any time from the Alerts page.

Searching arXiv

Search tips:

  • Use the basic search box for simple queries.
  • Use advanced search for field-specific queries.
  • Enclose phrases in "double quotes" for exact matches.
  • Use ? (single char) and * (multi char) for wildcards.
  • Separate multiple authors with a semicolon.

{% elif section == 'faq' %}

Frequently Asked Questions

Is arXiv peer-reviewed?
No. arXiv papers are moderated for scholarly merit and topical relevance, but are not peer-reviewed.
Do I need an account?
No — browsing and searching are free and open. An account is only needed for submitting papers or saving papers to your library.
How do I cite an arXiv paper?
Use the arXiv ID and year. A BibTeX snippet is provided on each paper's abstract page.
