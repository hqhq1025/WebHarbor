OAI-PMH — bulk metadata harvest interface

The arXiv archive exposes the OAI-PMH 2.0 protocol for bulk-harvesting paper metadata. This page describes the supported verbs, sets, and metadata formats. Endpoint base URL: http://export.arxiv.org/oai2.

Quick stats

  • Total records currently harvestable: {{ "{:,}".format(paper_total) }}
  • Supported metadata prefixes: arXiv, arXivRaw, oai_dc
  • Per-request throttle: 3.2 seconds — see arXiv's published rate-limit guidance.
  • Resumption token TTL: 1 hour (RFC-3339 timestamps).

Supported verbs

Verb Description
Identify Repository identity, admin email, granularity.
ListSets Enumerate available top-level archives.
ListMetadataFormats Available record schemas.
ListIdentifiers Header-only record listing with resumption tokens.
ListRecords Full metadata, paginated via resumption tokens.
GetRecord Single record by OAI identifier.
