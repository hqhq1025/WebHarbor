Command palette

Ctrl+K (or Cmd+K on macOS) opens a palette where you can paste any arxiv_id (e.g. 2401.12345) and press Enter to jump straight to /abs/<id>.

Listing navigation

On listings and search results, j moves to the next paper and k to the previous — the same vim-style muscle memory used by mu4e and Gmail.
