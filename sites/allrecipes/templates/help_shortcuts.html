Command Palette (Cmd+K)

The command palette opens a fuzzy search overlay across recipes, categories, collections, articles and static pages. Type any fragment and press Enter to jump. The palette is also accessible via the API at /api/command-palette/search?q=apple.

Contextual Help Popovers

Throughout the site, small [?] badges open inline popovers that explain the surrounding control. Press . (period) on any page to toggle the popover for the current section. A full inventory lives at /help/popovers.

Telemetry & Observability

Each keyboard-shortcut press fires a small JSON event to POST /telemetry so we can see which shortcuts get used. Read the most recent events back from /telemetry/events?n=5. Service health is published at /healthz and aggregate counters at /metrics (append ?format=prom for Prometheus text exposition).
