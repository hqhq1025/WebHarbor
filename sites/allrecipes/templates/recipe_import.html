Import via OCR — {{ recipe.title }}

Drop a photograph of a printed recipe into the upload zone and our OCR pipeline returns the parsed ingredient list + step list. You can also POST a JSON body with an ocr_text field directly to /recipe/{{ recipe.slug }}/import.

OCR Pipeline

  1. Upload or paste an image — JPEG / PNG / HEIC are supported.
  2. The image is rasterised to 300 DPI and passed through Tesseract 5.
  3. The raw OCR text is split line-by-line into ingredient (line begins with a digit or fraction) and step (everything else).
  4. The parsed structure is returned as JSON with a confidence score.

Example POST body

{
  "image_url": "https://example.com/scan.jpg",
  "ocr_text": "2 cups flour\n1 tsp salt\nWhisk dry ingredients...\nBake at 350F for 20 minutes."
}

Example response

{
  "slug": "{{ recipe.slug }}",
  "title": "{{ recipe.title }}",
  "ingredients": ["2 cups flour", "1 tsp salt"],
  "steps": ["Whisk dry ingredients...", "Bake at 350F for 20 minutes."],
  "confidence": 0.16
}
