Import recipe from URL

Paste any recipe URL and our importer parses it into a Mealie-compatible JSON document. You can also POST a JSON body with a url field directly to /recipe/import-url.

Import pipeline

  1. Fetch the recipe page over HTTPS with a browser-style User-Agent.
  2. Look for JSON-LD Recipe schema first; fall back to HTML scraping if absent.
  3. Normalize ingredients into a flat array and instructions into ordered HowToStep objects.
  4. Return the structured recipe as JSON with a confidence score.

Example POST body

{
  "url": "https://www.seriouseats.com/perfect-pan-pizza-recipe"
}

Example response

{
  "format": "mealie",
  "schema_version": "1.5",
  "source_url": "https://www.seriouseats.com/perfect-pan-pizza-recipe",
  "slug": "perfect-pan-pizza-recipe",
  "title": "Perfect Pan Pizza Recipe",
  "recipeYield": "4",
  "recipeIngredient": ["..."],
  "recipeInstructions": [{"@type": "HowToStep", "text": "..."}],
  "prepTime": "PT15M",
  "cookTime": "PT20M",
  "totalTime": "PT35M",
  "confidence": 0.72
}

Try it in your browser

(For programmatic use, send Content-Type: application/json and a JSON body. The HTML form posts as form-encoded; the parser will fall back to a stub response in that case.)
