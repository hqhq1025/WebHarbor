Developer API & integrations

REST, GraphQL, ICS calendar export, and webhooks. All endpoints are unauthenticated for the public read-only surface.

Endpoints

  • GET /api/v1/properties/search — REST property search
  • GET /api/v1/properties/search?min_rating=8 — with rating filter
  • POST /graphql — GraphQL endpoint (also accepts GET with ?query=)
  • GET /calendar/bookings.ics — ICS calendar export of your bookings
  • GET /api/v1/webhooks — registered booking-event webhooks
  • GET /api/v1/exchange-rates — currency rates

GraphQL example

{
  cities(limit: 5) {
    slug
    display
    country
    properties_count
  }
  properties(city: "paris", limit: 3) {
    name
    rating
    price
  }
}

Rate limits

1000 requests/hour per IP. Each response includes X-RateLimit-Remaining and X-Request-ID headers.
