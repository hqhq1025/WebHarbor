Accessibility at Coursera

We design Coursera to meet WCAG 2.1 Level AA across every learner-facing surface. This statement summarises our current compliance posture and how to report an issue.

WCAG 2.1 AA · CONFORMANT

Conformance summary

  • Perceivable — every video lesson ships captions in 11 languages; every image has alt text.
  • Operable — keyboard navigation across all interactive elements; visible focus rings; skip-to-main-content link on every page.
  • Understandable — UI strings translated into 16 locales (see footer switcher); reading level checked per category.
  • Robust — semantic HTML, ARIA labels on nav / search / dialog regions; tested with NVDA, JAWS, and VoiceOver.
