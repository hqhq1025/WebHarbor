Assistive-technology support

  • NVDA / JAWS / VoiceOver — every result card has an aria-label with title, source, snippet, and rank.
  • Skip-to-content link is the first focusable element on every page.
  • Knowledge panel facts use a definition-list (dl/dt/dd) so screen readers announce key→value pairs.
  • Search box honours prefers-reduced-motion; no autocomplete animation when set.

See also Advanced search syntax for operator-driven refinement that pairs well with keyboard navigation, per-operator glossary, and the Command palette (Cmd+K) for jump-to-anywhere navigation.
