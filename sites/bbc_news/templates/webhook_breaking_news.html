Breaking-news webhook

Subscribe a webhook URL to receive breaking-news push notifications from this BBC News mirror.

How to subscribe

POST /webhook/breaking-news
Content-Type: application/json

{
  "url": "https://your-host.example.com/hooks/bbc",
  "secret": "<shared-secret>",
  "events": "breaking-news,live-update"
}

Delivery contract

  • Topic: {{ data.topic }}
  • Format: {{ data.delivery.format }}
  • Retry policy: {{ data.delivery.retry_policy }}
  • Signature header: {{ data.delivery.signature_header }}

Confirm token

After a successful POST, respond 200 to a GET with ?challenge=<verify_token> to complete handshake.

Reference date: {{ data.reference_date }}. See also Developer API.
