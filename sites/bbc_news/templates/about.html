About the BBC

This is a learning mirror of the BBC News website, built as a standalone Flask application. It reproduces the visual design, content hierarchy, and common interactive features of the original, but contains no real BBC content. All articles, authors, and images are used for educational purposes.

What this mirror demonstrates

  • A news-site data model: articles, sections, topics, and user-generated comments.
  • A reading list, bookmarks, and exportable news digests.
  • Topic subscriptions and a per-user reading history.
  • User accounts with profile editing and password changes.

Tech stack

  • Flask + SQLAlchemy + SQLite for the backend.
  • Jinja2 server-rendered templates with vanilla JavaScript for interactivity.
  • Playwright-based scraper for harvesting hero images and page screenshots.
