Accessibility

Last updated: 4 March 2026.

We aim to make this BBC News mirror usable for everyone, regardless of ability. This page describes what we have done so far and how to get in touch if you find a barrier we have missed.

Standards

The site targets WCAG 2.2 Level AA. Pages are designed to be keyboard navigable, work at 200% zoom, and pass automated colour-contrast checks. Form fields carry labels, and landmark roles are used to mark up the header, navigation, main content and footer.

Known gaps

  • Some article hero images are still missing descriptive alt text.
  • The Live page does not yet expose status updates to screen readers.
  • The Reading List drag-and-drop reorder is not keyboard accessible.

Reporting a barrier

If something on the site does not work for you, please use the Contact page and choose the "Accessibility" subject so the message reaches the right person. We aim to respond within five working days.

Alternative formats

Articles can be exported as plain-text digests from the Digest page if you prefer to read without site styling.
