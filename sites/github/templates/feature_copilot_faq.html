General

What is GitHub Copilot?

GitHub Copilot is an AI pair programmer that helps you write code faster. It offers inline code suggestions in your editor as you type, a conversational chat interface for asking coding questions, and tools that help you understand unfamiliar code, generate tests, and review pull requests. Copilot is built on top of large language models from OpenAI and Anthropic, fine-tuned by GitHub on permissive open-source code.

How does Copilot work?

As you type, Copilot sends a snapshot of your current file (and a few related open files) to the Copilot service, which returns a ranked list of likely next-line completions. For Chat, your question plus a small relevant slice of your code is sent to the model, the model produces an answer, and that answer is streamed back into your IDE. None of these requests are stored on disk by default.

Which programming languages are supported?

Copilot works with every major language — JavaScript, TypeScript, Python, Java, Go, C, C++, C#, Ruby, Rust, PHP, Kotlin, Swift, Scala, Dart — plus configuration languages like YAML, Terraform, and SQL. Coverage is best for popular languages but generally usable for anything with substantial open-source code.

Which editors and IDEs does Copilot support?

Visual Studio Code, Visual Studio, the JetBrains IDE family (IntelliJ, PyCharm, WebStorm, GoLand, Rider, etc.), Neovim, and Xcode. There is also a CLI extension for terminal usage and a web experience on GitHub.com.

Pricing

How much does GitHub Copilot cost?

Copilot Individual is $10 USD per month or $100 USD per year. Copilot Pro+ is $39 USD per month for power users. Copilot Business is $19 USD per user per month for teams, and Copilot Enterprise is $39 USD per user per month with full org-wide policy controls. A free tier (Copilot Free) is available for everyone with 2,000 completions and 50 chat messages per month.

Is there a free trial?

Yes. New paid Individual subscribers get a 30-day free trial. After the trial, you can cancel any time and you'll keep access until the end of the billing period.

Is Copilot free for students or open-source maintainers?

Yes. Verified students and teachers get free access through GitHub Education, and maintainers of popular open-source projects get free access through the GitHub for Open Source program. Both programs include all features of Copilot Individual.

Mobile & Chat

Yes — Copilot Chat works on your phone. Here's the detail.

Copilot Chat in GitHub Mobile

The Copilot Chat experience is integrated into GitHub Mobile, supported on iPhone and iPad with iOS, and on phones and tablets running Android. Chat history syncs between desktop and mobile so you can continue a conversation from your laptop on your phone.

The mobile experience supports the same conversational queries as desktop — code explanations, bug-finding hints, repo Q&A, and natural-language search across your code.

Does Copilot Individual include access to Chat?

Yes. Copilot Individual includes Copilot Chat in supported IDEs, on GitHub.com, and in GitHub Mobile (both iOS and Android).

What can Copilot Chat do that completion can't?

Chat is a conversational interface — you ask a question in plain English (or any of 30+ supported languages) and Copilot answers, with optional code blocks. Use it to explain a function, generate tests for a file, suggest a refactor, write a regex, find the bug in a stack trace, or ask why a CI run failed. Completion is for "what comes next on this line"; Chat is for "how do I do X".

Can I @-mention specific files or repos in Chat?

Yes. In supported IDEs and on GitHub.com you can use #file, #selection, @workspace, or @github to scope the conversation to a specific piece of code or to your whole repository.

Privacy & Data

Does Copilot train on my private code?

No. Code from Copilot Individual, Business, and Enterprise customers is never used to train the public foundation model. Prompts and suggestions are also not retained for training by default. Enterprise customers can additionally opt out of any prompt retention for debugging.

Where does my code go when I use Copilot?

The minimum context required to produce a useful suggestion (typically the active file plus a few related files in your IDE workspace) is sent to the Copilot service over TLS, processed by the model, and discarded. Nothing is written to long-term storage in the default configuration.

Can Copilot suggest code that matches public open-source verbatim?

Rarely, but it can happen. Copilot includes an optional public-code filter that detects when a suggestion exactly matches more than 150 characters of public code on GitHub and either blocks the suggestion or shows you the matching repos and licenses so you can make an informed choice.

Enterprise

What's the difference between Copilot Business and Copilot Enterprise?

Both are seat-based licenses for organizations. Business adds policy management, license assignment, audit logs, and IP indemnity. Enterprise adds personalization on your private code (Chat answers are grounded in your repos), pull-request review skills, and a knowledge base feature for ingesting internal documentation.

Does Copilot offer IP indemnification?

Yes. Copilot Business and Enterprise customers receive standard IP indemnification for accepted suggestions when the public-code filter is enabled. See the GitHub Customer Agreement for the precise terms.

Can administrators control which features are enabled?

Yes. Org admins can disable Chat, completion, public-code matching, and individual Copilot extensions on a per-team or per-repo basis. Audit logs record every policy change.
