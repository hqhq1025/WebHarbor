Searching on GitHub

Use these search qualifiers to narrow your results when searching repositories. Combine multiple qualifiers in a single query — they all apply with AND semantics.

Repository qualifiers

QualifierExampleDescription
language: react language:typescript Match repositories written in a specific programming language. Languages are case-insensitive. Multi-word languages must be wrapped in quotes — for example language:"Common Lisp".
topic: topic:machine-learning Match repositories that have the given topic tag. Topic slugs are lowercase with hyphens (e.g. web-scraping, quantum-computing).
license: license:mit Filter by license SPDX identifier (mit, apache-2.0, gpl-3.0, bsd-3-clause, etc.).
archived: archived:false Either true or false. Archived repositories are read-only mirrors of older work.

Numeric qualifiers (stars, forks, size)

All numeric qualifiers support these comparison operators:

  • n — exactly n
  • <n — less than n
  • <=n — at most n
  • >n — greater than n
  • >=n — at least n
  • n..m — between n and m inclusive
QualifierExampleDescription
stars:react stars:>=10000Repositories with at least 10,000 stars.
forks:nlp forks:50..500Repositories with 50 to 500 forks.

Date qualifiers

Date qualifiers accept ISO-8601 dates (YYYY-MM-DD) and the same comparison operators as numeric qualifiers. Today's date in this site is {{ mirror_now.strftime('%Y-%m-%d') }}.

QualifierExampleDescription
created:created:>2024-01-01Repositories created after January 1st, 2024.
pushed:pushed:>2024-05-08Repositories with commits pushed after this date — useful for "recently active" searches.
updated:updated:>2024-05-08Alias of pushed:.

Sort qualifier

Use sort: in your query, or pick from the "Sort by" dropdown on the search results page.

ValueDescription
best-matchDefault. Ranks by query-token relevance, then by stars.
stars / stars-ascMost / fewest stars.
forks / forks-ascMost / fewest forks.
updated / updated-ascMost / least recently updated.

Putting it together

Find recently active Python ML repositories with substantial adoption:

machine learning language:python stars:>=500 pushed:>2024-05-08

Find newly-created blockchain projects:

blockchain created:>2024-01-01 sort:stars

Find C++ game-development repositories that aren't archived:

game-dev language:c++ archived:false

Tips

  • Wrap multi-word phrases in quotes to match them exactly: "decision tree" language:python.
  • Qualifiers are AND-combined. There is no OR operator.
  • Negate a qualifier with a leading minus: -language:javascript excludes JavaScript repos.
  • Most qualifiers are also exposed as URL parameters (?language=python&min_stars=500&updated_days=7) for direct linking.
